# Use Cases

This section contains step-by-step walkthroughs for the most common UranieMUSC workflows.

## Which use case should I start with?

- **First time running UranieMUSC?** Start with {doc}`full_run` — it walks through every step from a fresh checkout to NetCDF output.
- **Running a sensitivity study?** See {doc}`sensitivity_morris` for the Morris Elementary Effects method, which is the default setup.
- **Want random sampling instead?** See {doc}`lhs_sampling` for Latin Hypercube Sampling.
- **Adding a new parameter to perturb?** See {doc}`add_parameter`.
